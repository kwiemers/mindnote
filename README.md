# mindnote
 
A personal mood and journal tracking web app — built from scratch as a self-directed learning project.
 
## About
 
mindnote is a small full-stack web application that lets you log your daily wellbeing and write free-form journal entries. The home page displays your most recent journal entry and calculates the average values of your tracked mood data across all entries.
 
## Tech Stack
 
- **PHP** – server-side logic, form handling and database communication
- **MySQL** – relational data storage via PDO with prepared statements
- **HTML/CSS** – semantic markup, responsive layout and custom design
- **JavaScript** – lightweight client-side interactions
 
## Features
 
- Daily tracking form with 5-point rating scale across four categories (mood, sleep, emotional and physical wellbeing)
- Free-form journal entries with persistent storage
- Home dashboard showing the latest journal entry and average tracking values
- Input validation with user feedback
- SQL injection protection via prepared statements and output escaping (XSS)
